# 💡 Tech Event English Phrases - Flashcards

Aplicação web simples para praticar de forma interativa frases básicas/essenciais em inglês para networking em eventos de tecnologia.

## 💻 Visão Geral

Aplicação de página única (SPA) que exibe um conjunto de flashcards com frases comuns em inglês utilizadas em contextos profissionais e eventos de tecnologia. Cada cartão apresenta uma frase, seu contexto de uso, explicação/tradução e um exemplo prático. O objetivo é ajudar falantes não nativos a se sentirem mais confiantes ao se comunicar em eventos internacionais.

## ⚒️ Funcionalidades

Flashcards Interativos: Clique ou toque para virar o cartão e ver explicações e exemplos.
Categorias: Cada frase é agrupada por contexto (Apresentações, Conversa Informal, Networking, etc.).
Controles de Navegação: Avance, retorne ou embaralhe os cartões para praticar de forma aleatória.
Indicador de Progresso: Mostra sua posição atual no baralho.
Design Responsivo: Funciona bem em computadores e dispositivos móveis.
Atalhos de Teclado:
Setas esquerda/direita: navegação entre cartões
Barra de espaço: virar o cartão
